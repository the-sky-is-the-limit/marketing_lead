# マーケティング分析ダッシュボード

ヘッジファンドダイレクトのオンライン集客データを5つの視点から分析するStreamlitアプリです。

## ページ構成

| ページ | 内容 |
|--------|------|
| ① エグゼクティブサマリー | 全体ファネル・KPI・月次推移 |
| ② 単軸ファネル分析 | リードソース/資産/年代/経験/職業別の転換率比較 |
| ③ 二軸クロス分析 | 任意2変数のヒートマップ |
| ④ 多軸ドリルダウン | 3変数までのインタラクティブ深掘り |
| ⑤ リードスコアリング | 属性組み合わせによる期待成約率シミュレーション |

## セットアップ

### ローカル実行

```bash
pip install -r requirements.txt
streamlit run app.py
```

### Streamlit Community Cloud

1. このリポジトリをGitHubにプッシュ
2. Excelファイル（`マーケデータクレンジング２.xlsx`）をリポジトリ直下または `data/` フォルダに配置
3. [share.streamlit.io](https://share.streamlit.io) からデプロイ

> **注意:** Excelファイルがリポジトリに含まれていない場合、サイドバーのファイルアップローダーからアップロードできます。

## リポジトリ構成

```
├── app.py                          # メインアプリ
├── requirements.txt                # 依存パッケージ
├── .streamlit/
│   └── config.toml                 # テーマ設定
├── マーケデータクレンジング２.xlsx     # データファイル（任意）
└── README.md
```
